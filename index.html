Education and Achievements


Bachelors's Hounours in Computer Systems Dissertation Project - Interaction (Tech Demo): Beyond Souls

Created a game from scratch using the Unity3D game engine to conduct a systematic analysis on a game’s user interface, included with its overall performance, as well as player’s behaviour in a stressful and eerie virtual environment, using volunteers who are familiar with common video-game controls. The project involves studying, as well as play-testing, different multiplayer games in order to gain a first-hand perspective to hypothesize a core relationship between player and an interaction.

Production of a multiplayer game, where users can interact with one another, as well as, the objects that surround them is vital; especially in acquiring the analysis to the aforementioned relationship. The game will be bare-bones, the mechanics will be solely for interaction purposes. A qualitative questionnaire will be handed to the students after completing a round within the game..
