Education and Achievements
















Bachelor's of Science in Computer Systems with 2:1 Honours from Heriot-Watt University (Edinburgh, UK).

Acquired a 2:1 (Upper Division) Honourary Degree over the course of 4 years. A summary of my most notable achievements over my academic career (2014-2018):

• Dissertation Project: Focused on the field of Interaction in Video Games (Tech Demo: Beyond Souls): Created a game from scratch using the Unity3D game engine to conduct a systematic analysis on a game’s user interface, included with its overall performance, as well as player’s behaviour in a stressful and eerie virtual environment, using volunteers who are familiar with common video-game controls. The project involves studying, as well as play-testing, different multiplayer games in order to gain a first-hand perspective to hypothesize a core relationship between player and an interaction. Production of a multiplayer game, where users can interact with one another, as well as, the objects that surround them was vital; especially in acquiring the analysis to the aforementioned relationship.

• WP (Project 2) - Created a Twitter Follower Manager website, using the Twitter API implementing AJAX and MySQL databases.

• Hardware-Software Interface (Project 1) - Created a steganography software using C to be used with “.PPM” format files using a 2D array.

• HSI (Project 2) - Recreated the game "Mastermind", using C and assembly language.

• Interaction Design - Designed a full-scaled penguin robot caretaker "Pengubot". As a team we carried on an experimental protocol and demonstrated a video/visual strip of a scenario.

• Web Design and Databases - Created a fully functional game store website "Planet Undead" to showcase our skills of web design including PHP and MySQL databases. Awarded the best website theme and design.

• Computer Network Security (Project 1) - Symmetric Encrypting Algorithm: Completed taaks in learning core concepts of secret key encryption, by obtaining evidence-based experience when encrypting, as well as, decrypting messages.

To top it all off, we implemented a smart learning algorithm that produced supervised inductive learning method to adapt and recognize a credit/debit card.

We needed to leverage at least one “VISA Payment Method API” to create a digital payment application. We competed against 19 teams. After developing for 24-hours, we created a fully functional augmented reality application called “VISA AR” and emerged victorious, placing second.
